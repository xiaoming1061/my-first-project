## ✨ 功能 / 学到了什么
- ✅ 用 HTML 搭建网页骨架
- ✅ 用 `h1`、`p`、`a` 等标签组织内容
- ✅ 用 Git 管理代码版本（`init` → `add` → `commit`）
- ✅ 把本地代码推送到 GitHub（`push`）
- ✅ 理解 Git 和 GitHub 的区别
- ✅ 修改文件后重新提交，体验完整的 Git 工作流

## 🚀 如何运行
1. 克隆这个仓库到本地：
   ```bash
   git clone https://github.com/你的用户名/my-first-project.git
   ```
2. 用浏览器打开 `index.html`
3. 或者右键文件 → **Open with Live Server**（如果你装了 VS Code 的 Live Server 插件）
